# Markdown-навигатор NVDA

Markdown-навигатор — это дополнение для NVDA, спроектированное для улучшения работы с текстом в формате Markdown. Оно позволяет использовать функции навигации с помощью одной клавиши, аналогичные режиму обзора NVDA, в редактируемых текстовых областях (например, в Блокноте, VS Code, веб-редакторах и т.д.).

Оно позволяет быстро переходить к заголовкам, спискам, таблицам, блокам кода и различным строчным форматированиям без частого нажатия клавиш со стрелками.

## Ключевые особенности

*   **Эффективная навигация**: Реализует быстрые алгоритмы, позволяющие выполнять мгновенные переходы даже в больших документах, содержащих десятки тысяч строк.
*   **Структурированный просмотр**: Поддерживает навигацию по заголовкам, спискам, таблицам, цитатам, блокам кода и многому другому.
*   **Строчные элементы**: Поддерживает переход к выделению полужирным шрифтом, курсивом, ссылкам, изображениям и строчным вставкам кода.
*   **Навигация по ячейкам таблицы**: В таблицах Markdown предоставляет сочетания клавиш, соответствующие стандартным командам NVDA.

## Использование

### Переключение режима обзора
По умолчанию навигация одной клавишей отключена. Вам необходимо вручную включить режим обзора Markdown.

*   **Сочетание клавиш**: `NVDA + Shift + Space` (Пробел)
*   **Эффект**: Включает или выключает функциональность дополнения. NVDA объявит "Режим обзора Markdown включён/выключен". Эта функция также соотносится с настройкой NVDA "Звуковая индикация режимов обзора и редактирования" в настройках режима обзора.

### Список сочетаний клавиш (при включённом режиме обзора)

После включения режима вы можете использовать следующие однобуквенные клавиши.
*   Нажмите **буквенную клавишу**, чтобы перейти к следующему элементу.
*   Нажмите **Shift + Буквенную клавишу**, чтобы перейти к предыдущему элементу.

| Клавиша | Тип элемента | Описание |
| :--- | :--- | :--- |
| **H** | Заголовок | Заголовок любого уровня (от `#` до `######`) |
| **1-6** | Заголовок 1-6 | Конкретный уровень заголовка |
| **T** | Таблица | Начало таблицы |
| **L** | Список | Начало блока списка |
| **I** | Элемент списка | Конкретный элемент списка (`-`, `*`, `1.`) |
| **Q** | Цитата | Блок цитаты (`>`) |
| **C** | Код | Блок кода (```` ``` ````) или строчная вставка кода (`` ` ``) |
| **S** | Разделитель | Горизонтальная линия (`---`, `***`) |
| **X** | Флажок | Элемент списка задач (`- [ ]`, `- [x]`) |
| **K** | Ссылка | Ссылка Markdown `[текст](url)` |
| **G** | Графический элемент | Тег изображения `![alt](url)` |
| **B** | Полужирный | Полужирный текст (`**` или `__`) |
| **E** | Акцент | Курсивный/акцентированный текст (`*` или `_`) |
| **D** | Удаление | Зачёркнутый текст (`~~`) |
| **F** | Ссылка | Ссылка на сноску [^1]` |
| **,** | Конец блока | Перейти к концу текущего блочного элемента |
| **Shift+,**| Начало блока | Перейти к началу текущего блочного элемента |
| **M** | Математика | Математическое выражение LaTeX (`$...$` или `$$...$$`) |

### Навигация по ячейкам таблицы

Когда курсор находится внутри таблицы Markdown и включён режим обзора, вы можете использовать следующие сочетания клавиш для перемещения между ячейками:

*   **Ctrl + Alt + Стрелка влево**: Предыдущая ячейка
*   **Ctrl + Alt + Стрелка вправо**: Следующая ячейка
*   **Ctrl + Alt + Стрелка вверх**: Ячейка выше
*   **Ctrl + Alt + Стрелка вниз**: Ячейка ниже

## Примечания

* Это дополнение разработано специально для сценариев редактирования Markdown. Если вы читаете HTML-страницу, отображаемую браузером, используйте встроенный режим обзора NVDA.
* Если нажатие указанных выше клавиш вводит символы вместо навигации, проверьте, включён ли режим обзора, нажав `NVDA + Shift + Space`.
* В VS Code, из-за того, что редактор отображает содержимое по запросу (виртуализация), режим обзора этого дополнения может работать некорректно для содержимого, которое в данный момент не видно/не отображается.
